# Requirements

As an anonymous user, I want to
1. See all conferences
2. Search for conferences (by name, by author)
3. See detailed information about a conference
4. Be able to see the link to a conference (and navigate to it)
5. Sign Up/Sign In using my existing Google or GitHub account

As signed in/logged in user, I want to
1. Do everything an anonymous user can
2. Sign Out
3. Add a new conference to the database
   * Add a name - required
   * Add a link - required
   * Add a description - optional
   * Add a speaker - required
   * Add tags - optional
   * Add organizer - optional
4. Delete my added conferences
5. Update my added conferences
6. Favorite a conference talk (bookmarks)
7. Unfavorite a conference talk
8. View all favorite conference talk list
